## Quick Tips

### Choosing the Right Scope

- Use `target: project` for team-shared rules and project-specific configurations
- Use `target: user` for personal preferences and global workflows

### Common Workflows

```bash
# Deploy to project (default)
calvin deploy

# Deploy to home directory
calvin deploy --home

# Preview changes before deploying
calvin diff

# Watch for changes and auto-deploy
calvin watch

# Check configuration for issues
calvin check

# Clean deployed files (interactive)
calvin clean

# Clean all home deployments
calvin clean --home --yes
```

### Troubleshooting

If you encounter issues:

1. Run `calvin check` to validate your configuration
2. Use `calvin diff` to preview what would change
3. Check the [Pitfall Mitigations](/guides/pitfall-mitigations) guide for common issues
